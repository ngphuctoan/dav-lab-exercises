# Data Analysis and Visualisation &mdash; Lab Exercises

A collection of lab exercises exploring statistics, hypothesis testing, regression, and visualisation using Python, Jupyter Lab, and various datasets.

<a name="installation"></a>
## Installation

Requirements:

- Python 3.13+  
- [uv package manager](https://github.com/Astral-uv/uv)

Install all dependencies:

```bash
uv sync
```

<a name="usage"></a>
## Usage

Start a Jupyter Lab server:

```bash
uv run --with jupyter jupyter lab
```
